 Labour Market Trends Dashboard (2021–2025)
This project presents an interactive Labour Market Trends Dashboard built using Statistics Canada datasets. It analyzes employment patterns across Canada from 2021 to 2025, highlighting key economic and demographic shifts during this period.
The dashboard was created as part of an academic project for NSCC (2025) and is designed to support data‑driven insights for decision‑makers, educators, and workforce planners.

📌 Project Overview
This dashboard provides a clear and interactive way to explore labour market data, including:

Employment & unemployment trends over multiple years
Industry‑level growth and decline
Demographic patterns (age groups, gender distributions, regional variations)
Year‑over‑year comparisons to identify changes and emerging labour market signals

Users can filter and drill down into specific areas to reveal underlying trends and relationships across multiple demographic and economic dimensions.

🛠️ What I Did

Built an interactive Excel dashboard using PivotTables, slicers, charts, and Power Query.
Imported and transformed Statistics Canada datasets, cleaning and organizing raw data for analysis.
Performed data cleaning and harmonization, including fixing data types, handling missing values, and standardizing fields.
Created visualizations to highlight employment indicators, industry movements, and demographic shifts.
Designed the dashboard layout to ensure clarity, accessibility, and ease of interpretation for stakeholders.


🎯 Purpose of the Dashboard
The main aim of this project is to provide an accessible reporting tool that:

Supports evidence‑based decision‑making
Helps users quickly identify critical labour market changes
Makes large datasets easier to understand through interactive visual exploration
Provides a snapshot of economic recovery patterns, industry transformations, and workforce dynamics from 2021 to 2025
